# Turtle Patterns

A growing collection of creative patterns and generative art created using Python's Turtle graphics.

Each pattern is organized in its own directory and includes:

* Python source code (`.py`)
* A preview image (`.png`)
* A dedicated `README.md` with project details

## Patterns

| Pattern                | Preview                                             |
| ---------------------- | --------------------------------------------------- |
| Concentric Arc Pattern | `concentric_arc_pattern/concentric_arc_pattern.png` |

More patterns will be added over time.

## Requirements

* Python 3.x
* Turtle (included with the standard Python installation)

## Run a Pattern

Navigate to the desired pattern directory and run its Python file.

```bash
python concentric_arc_pattern/concentric_arc_pattern.py
```
